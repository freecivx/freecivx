# FreecivX 3D Java jMonkeyEngine Client

The **FreecivX Java jMonkeyEngine Client** is the client application to play Freeciv on Freecivx servers. 

https://github.com/jMonkeyEngine/jmonkeyengine

https://jmonkeyengine.org/

Goal:
=====
Freecivx in 3D on Java.



## Repository Structure
This repository contains:
- **freecivx-server**: The backend server that powers the FreecivX game logic.
- **freecivx-client-3d**: The Java Swing client that connects to the server and provides the user interface.

## Requirements
- Java 21 or later

## Quick Start
1. Build the Swing client:
   ```bash
   cd freecivx-client-3d
   mvn clean package
   ```

2. Run the Swing client:
   ```bash
   java -jar target/freecivx-client-3d-1.0.0.jar
   ```

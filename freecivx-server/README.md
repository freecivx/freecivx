Freecivx Server Java
====================

freecivx-server is a multiplayer server implemented in Java for Freecivx.net.

 - Multiplayer games focus.
 - Large MMO-games
 - Many players
 - Very large maps
 - Long-running games
 - Use modern features from Java 21, such as virutal thrads.
 - Websockets
 - JSON or Protobuf protocol
 - No AI yet.


Requires Java 21 or later. Build with maven.

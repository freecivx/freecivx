Freeciv C server 
----------------

This is the forked Freeciv C server.

**prepare_freeciv.sh** - A script which will checkout Freeciv from Git, apply the patches and finally configure and compile the Freeciv C server.

**compile_freeciv_v86.sh** - A script to compile Freeciv as a static x86 executable for use with v86 emulator. See `freeciv-v86-README.md` for details.

**version.txt** - Contains the Git revision of Freeciv to check out from Git.
